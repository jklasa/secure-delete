# secure-delete

Quick implementation of a secure delete tool that can have configurable overwrite methods, numbers of passes, and buffer sizes

#### Possible commands for testing

./sdel file
./sdel file -b 28
./sdel file -s 5

./sdel file -t 2
./sdel file -t 2 -s 8

./sdel file -t 3
./sdel file -t 3 -s 5
